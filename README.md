# Calculator Using Tkinter

A simple Calculator application built using Python and Tkinter. This project provides a user-friendly graphical interface for performing basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

* Clean and interactive GUI using Tkinter
* Basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
  * Equals
* Clear button to reset input
* Displays results instantly
* Easy-to-understand and beginner-friendly code

## Technologies Used

* Python
* Tkinter (Python GUI library)

## Installation

1. Clone this repository:

```bash
git clone https://github.com/your-username/calculator-tkinter.git
```

2. Navigate to the project folder:

```bash
cd calculator-tkinter
```

3. Run the Python file:

```bash
python calculator.py
```

## Requirements

Make sure Python is installed on your system.

Check Python version:

```bash
python --version
```

Tkinter comes pre-installed with most Python distributions.

## How It Works

The calculator takes input through GUI buttons created using Tkinter widgets. User actions trigger functions that process expressions and display the calculated result.
